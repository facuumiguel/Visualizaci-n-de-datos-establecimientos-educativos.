El objetivo principal de este trabajo es
investigar si existe algún patrón o relación entre la cantidad de Establecimientos Educativos
(EE) y la de Bibliotecas Populares (BP) en las diferentes provincias y departamentos de
Argentina, usando datos oficiales del censo 2022. Para ello seguimos los siguientes pasos:
1. Revisión y limpieza de datos: examinamos la calidad de los datasets de EE y BP,
aplicando el enfoque GQM (Goal–Question–Metric) para cuantificar problemas de
completitud e inconsistencia.
2. Modelado: diseñamos un DER en 3FN y su correspondiente modelo relacional.
3. Preparación: Con Python, Pandas y SQL limpiamos y creamos las respectivas tablas
guiándonos de nuestro modelo relacional.
4. Consultas y visualizaciones: Mediante SQL, Pandas y Matplotlib obtuvimos tablas
que resumen cantidades de EE y BP por departamento, así como visualizaciones
que muestran la distribución y una posible relación entre ambas variables.
